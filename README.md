# README
## Starting
You may run this on any device and may even download soon Electron apps for offline executions
#### Fullscreen
Press F11, then connect to screen/TV and you can play now
## Controller support
This system supports DualSense: Click the DualSense option at start
This system supports DualShock: Click the dualshock option at start
This system does ***NOT*** support nintendo, atleast my coding lang doesn't
To use other controllers (which *may* not work) Use the dualshock option at the start
## Features (so far)
This project is a **beta** and is not perfectly able to be used as a normal console system.
### Settings
Option to change quality.
### Cursor
On DualSense, press the big rectangle button in the middle (i don't know what to call it) on your controller.
On DualShock/Other, Press the down arrow key on your controller.
